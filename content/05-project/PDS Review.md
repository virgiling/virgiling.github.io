# Review 意见详细总结

## 论文基本信息

- **论文标题**: An Improved ILP Model and Multi-start Iterated Local Search for the Power Dominating Set
- **期刊**: Computers and Operations Research
- **稿件编号**: CAOR-D-25-01422
- **决定**: 拒绝发表

## Reviewer #1 主要意见

### 1. 研究动机问题

- **核心批评**: 在密集图上研究 PDS 问题的动机不够充分
- **具体问题**:
  - 电力网络图通常是稀疏图，密集图上的 PDS 研究缺乏实际意义
  - 没有给出在密集网络上 PDS 问题的有意义的解释
  - 传播性质在密集图中变得不那么相关，支配性质可能更为重要

### 2. ILP 模型缺陷

- **模型创新性不足**:

  - 对文献[14]和[10]中的模型只有微小改动
  - 约简规则的自适应版本在[10]中已经存在

- **模型技术问题**:
  - 提出的 ILP 模型有严重缺陷：没有防止被排除的顶点被选中
  - 约简规则选择不明确，只考虑低度顶点，这在密集图中很少见

### 3. 混合方法设计问题

- **当前方法的问题**:

  - 基于固定参数选择，不是真正的混合方法
  - 没有充分利用两种方法的优势

- **建议的改进方向**:
  - 使用启发式解来引导 ILP 求解器
  - 用 ILP 的中间解来改进启发式的上界
  - 实现真正的协同工作模式

### 4. 算法实现问题

- **算法 5 第 4 行错误**:
  - 计算的是平均顶点度的一半，而不是图密度
  - 需要修正为正确的图密度计算

### 5. 实验评估问题

- **比较方法不当**:

  - 比较具有不同目标的方法（启发式 vs 精确方法）具有误导性
  - 精确方法也产生下界和中间启发式解

- **评估指标缺失**:
  - 应该显示各种启发式解与最优解的接近程度
  - 当精确方法超时时，应至少提供中间解的值和下界
  - "NA"条目的含义不明确（超时还是其他原因失败）

## Reviewer #2 主要意见

### 1. 写作和结构问题

- **整体质量问题**:

- **贡献说明不明确**:

  - 需要清晰界定在模型和算法方面的具体进展

- **关键文献缺失**:

  - 最新的方法应该包含在计算比较中
